   Get the application URL:

    echo "Tracefinity is available at: $APP_URL"

{{- else if contains "NodePort" .Values.service.type }}
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "tracefinity.fullname" . }})

  export CONTAINER_PORT=$(kubectl get pod --namespace {{ .Release.Namespace }} $POD_NAME -o jsonpath="{.spec.containers[0].ports[0].containerPort}")
  kubectl --namespace {{ .Release.Namespace }} port-forward $POD_NAME 8080:$CONTAINER_PORT &
  echo "Tracefinity is available at: http://127.0.0.1:8080"
{{- end }}

2. Getting started:
   - Place your tools on A4/Letter paper and take a photo from above
   - Upload the photo and adjust the paper corners for scale calibration
   - AI will trace the tool outlines automatically
   - Save traced tools to your library and arrange them into bins
   - Export STL/3MF files and print your custom Gridfinity bins

3. Tracing modes:
   - Local model (default, no API key needed): uses IS-Net, requires ~2GB RAM
   - Gemini API: set GOOGLE_API_KEY for higher accuracy tracing
   - Check your values with: kubectl get --namespace {{ .Release.Namespace }} deployment {{ include "tracefinity.fullname" . }} -o jsonpath="{.spec.template.spec.containers[0].env}"

{{- if .Values.persistence.enabled }}
4. Storage: tool library and project data is persisted to a PersistentVolumeClaim.
   Check storage: kubectl get --namespace {{ .Release.Namespace }} pvc {{ include "tracefinity.fullname" . }}-data
{{- else }}
4. WARNING: persistence is disabled. Your tool library and project data will be lost if the pod restarts.
   Enable it with: --set persistence.enabled=true --set persistence.size=1Gi
{{- end }}

For docs and guides: https://github.com/tracefinity/tracefinity
